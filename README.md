# OpenCV Video/Photo Face Detection
OpenCV를 이용하여 영상에서 얼굴을 인식하는 프로젝트 예제입니다.

## 시작하기
### 의존성 패키지 설치
이 예제는 OpenCV 의존성 패키지를 사용합니다. 아래 명령을 사용하여 의존성 패키지를 설치하세요.

```
pip install -r requirements.txt
```

### 프로젝트 실행
아래 명령을 실행하면 프로젝트를 실행할 수 있습니다.  
```
python main.py
```

## 라이선스
